## 📁 Project Index

### Final Year Project
**[FYP Network Security Tool](./FYP%20Network%20Security%20Tool)**
`Python` `Scapy` `PySNMP` `VirusTotal API` `CustomTkinter`

End-to-end LAN discovery and security auditing desktop tool. Performs ARP/ICMP host
discovery, SYN port scanning, TTL-based OS fingerprinting, and OUI/MAC vendor lookup.
Integrates VirusTotal for IP reputation checks and generates SHA-256-verified HTML audit
reports aligned with ISO/IEC 27001, ITIL v4, and PDPA 2010.
